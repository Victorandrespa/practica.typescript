Que es Typescript?

TypeScript es un lenguaje de programación desarrollado por Microsoft que extiende a JavaScript agregándole tipado estático (definir el tipo de una variable, parámetro o función) y otras características útiles para el desarrollo de aplicaciones a gran escala. TypeScript te permite especificar los tipos de datos (como string, number, boolean, array, etc.), lo cual ayuda a evitar errores comunes desde la etapa de desarrollo.

Ventajas de typescript frente a javascript puro

Tipado estatico
  • Declara tipos de datos
  • Detecta errores en tiempo de desarrollo
  • Mejora la autocompletacion y documentacion automatica en los editores de codigo
Mejor mantenimiento del codigo
  • Hace que el codigo sea mas legible y facil de mantener, especielmente en equipos grandes
  • Ayuda a entender como interactuan los distintos componentes
Compilable con javascript
  • Typescript es como un superset de Javascript, lo que significa que todo el codigo Javascript es valido en Typescript
  • Se puede migrar un proyecto poco a poco a TS
Mejor soporte en editores
  • Autocompletado mas inteligente
  • Navegacion facil
  • Documentacion emergente y sugerencias
Mejora la integracion de Frameworks modernos
  • Frameworks como Angular estan escritos en typescript
Deteccion temprana de errores
  • Compilador (tsc) te avisa antes de que el codigo se ejecute, reduciendo errores en produccion
Como se instala Typescript en VS code
  1. Descargar e instalar node.js desde la pagina oficial
  2. Abrir terminal y verificar instalacion “node -v” , “npm -v”. Deben aparecer las versiones
  3. En terminal escribir “npm install -g typescript”
  4. Verificar instalacion con “tsc -v”
  5. Iniciar un proyecto y crear una funcion de prueba
  6. En la terminal de vs code agregar “tsc –init”, esto crea un archivo tsconfig.json
  7. Compilar el archivo “tsc”, esto genera un archivo index.js
  8. En la terminal “node index.js, deberias ver el resultado.
Como se compila un archivo Typescript para convertirlo en Javascript

Los navegadores no entienden TypeScript directamente, por eso se transpila (convierte) a JavaScript.

Usas tsc (TypeScript Compiler) para generar archivos .js a partir de .ts.

Se define usando : seguido del tipo despues del nombre de la variable, parametro o funcion

Variables: Let nombre: string = “juan”;

Arreglos: Let numeros: numer [] = [1, 2, 3];

Objetos

Que son los tipos de union y como se declaran

Los tipos de unión (union types) en TypeScript te permiten declarar una variable que puede ser de más de un tipo. Se declaran usando el símbolo | (pipe) entre los tipos posibles.

Que es un tipo literal en Typescript y para que se utiliza

En TypeScript, un tipo literal es un tipo que representa valores específicos y exactos, no solo un tipo general como string o number, sino un valor concreto, como "activo" o 3.14.

Se usa para: Restringir valores posibles, crear tipos mas seguros y predecibles, validar de forma estricta que solo se usen ciertos valores.

Que son los tipos enumerados (Enums) y como se crean

Los tipos enumerados o Enums en TypeScript son una forma de definir un conjunto de valores con nombre, que representan un grupo fijo de constantes relacionadas. Sirven para manejar valores simbólicos legibles en lugar de usar números o strings sueltos.

  • enum Estado{
  • Activo,
  • Inactivo,
  • Pendiente
  • }
Como se define un tipo alias y cuando es util

Un tipo alias en TypeScript es una forma de darle un nombre a un tipo, ya sea un tipo primitivo, un objeto complejo, una unión, o cualquier combinación de tipos. Se define con la palabra clave type.

  • type NombreDelTipo = tipo;
  • type Edad = number;
  • let miEdad: Edad = 30;
Como se definen las funciones con tipos en typescript

En TypeScript, puedes definir funciones con tipos para los parámetros y el tipo de retorno, lo que ayuda a que el código sea más seguro y claro.

function sumar (a: number, b: number): number

Que son los parametros opcionales y los parametros predeterminados en funciones
Parámetros Opcionales
  • Son parámetros que pueden o no recibirse al llamar a la función
  • Se indican agregando un signo de interrogación ? después del nombre del parámetro.
  • Si no se pasa el argumento, el valor será undefined dentro de la función.
Parametros Predeterminados (Default Parameters)
  • Son parametros que tienen un valor por defecto si no se proporciona uno al llamar la funcion
  • Se definen asignando un valor directamente en la firma de la funcion.
  • Si no se pasa el argumento, se usa el valor por defecto.
